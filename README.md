# Blooms and Petals Website

Blooms and Petals is a multi-page floral boutique website created using HTML and CSS. The project showcases a structured layout with dedicated pages for products, collections, and customer interaction.

## Pages Included
- Home
- About
- Collection
- Gallery
- Contact
- Sign In

## Features
- Multi-page navigation system
- Custom CSS styling
- Organized content sections
- Image-based gallery and product display

## Technologies Used
- HTML5
- CSS3
